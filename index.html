Calculadora de Status: Ordem Paranormal RPG

Este projeto é uma calculadora interativa desenvolvida para o sistema de RPG Ordem Paranormal, com o objetivo de agilizar a criação e o balanceamento de personagens. A aplicação permite que o usuário insira os cinco atributos principais (Força, Vigor, Inteligência, Presença e Agilidade) e, com base neles e na classe do personagem (Combatente, Especialista ou Ocultista), a calculadora gera automaticamente os valores de Vida, Defesa e Pontos de Esforço, seguindo as regras oficiais do sistema.

A calculadora foi inicialmente desenvolvida em Python puro e posteriormente migrada para Streamlit, proporcionando uma interface gráfica amigável, acessível via navegador, facilitando o uso durante sessões de jogo ou criação de fichas.

Funcionalides Principais

  • Entrada personalizada dos atributos do personagem.
  • Seleção da classe do personagem para cálculo automático de atributos secundários.
  • Geração automática de valores de Vida, Defesa e Pontos de Esforço.
  • Interface visual intuitiva com Streamlit.
  • Aplicação leve, rápida e acessível via web.

Tecnologias utilizadas:

  • Python
  • Streamlit

💡 O projeto seguirá sendo atualizado até comportar não somente o sistema de RPG Ordem Paranormal, mas também outros como D&D, Tormenta, Call of Cthulhu etc.

Controle de Iluminação via Interface Web

Este projeto está sendo desenvolvido por alunos do primeiro período do curso de Análise e Desenvolvimento de Sistemas, com o objetivo de aplicar na prática os conhecimentos iniciais de hardware e desenvolvimento web. A proposta é permitir o controle da iluminação de uma sala real por meio de uma interface web responsiva, utilizando Arduino e um módulo relé.

O sistema conecta o Arduino a uma rede local, permitindo que o usuário acesse uma página web simples e acione ou desligue as luzes com apenas um clique. O projeto tem foco em acessibilidade, baixo custo e na integração entre software e hardware.

Funcionalidades principais:

  • Interface web simples para acionamento de lâmpadas.
  • Comunicação entre página web e Arduino via rede local.
  • Controle em tempo real dos estados (ligado/desligado).

Tecnologias utilizadas:

  • HTML, CSS e JavaScript para a interface web.
  • Arduino para controle de hardware.
  • Módulo relé para acionamento das lâmpadas.

💡 O projeto está em andamento e seguirá sendo atualizado até que possa controlar não somente a iluminação, mas também ligar e desligar os computadores da sala.

Robô Jacaré: Projeto de Automação e Sensores

Este projeto consiste na criação de um robô móvel com a aparência de um jacaré, desenvolvido com o objetivo de aplicar conceitos básicos de robótica, sensores e automação. O robô possui um sensor de proximidade instalado em sua “boca”, que permite detectar objetos ou pessoas em seu campo de alcance.

Quando o sensor identifica um objeto dentro de um determinado raio, o robô ativa seu sistema de locomoção para perseguir o alvo, simulando o comportamento de um jacaré que persegue presas.

Funcionalidades principais:

  • Detecção de objetos por sensor de proximidade.
  • Movimentação automática do robô para seguir o objeto detectado.
  • Sistema embarcado simples para integração dos sensores e motores.

Tecnologias e componentes utilizados:

  • Microcontrolador.
  • Sensor de proximidade.
  • Motores DC com driver.
  • Programação em C/C++ para controle do robô
